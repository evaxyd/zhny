数据可视化

核心：让数据更易读，更直观。

目的：

1、分析问题，得出结论，以供人决策参考 2、展现现象，把复杂的问题轻松化、直观化、简单化 3、预测未来，做趋势性判断

一个好的数据可视化作品，一定是集美观、数据价值、可读性等一体的，一定是让信息变得简单。

厉害的Echarts

  • 一个纯 Javascript 的图表库
  • 兼容主流浏览器，移动友好
  • 底层依赖轻量级的 Canvas 类库 ZRender
  • 提供直观、生动、可交互，可高度个性化定制的数据可视化图表

一步一步来

1、获取（官网、github、npm、cdn） 在webpack中使用

2、将echarts引用到你的页面中

3、创建一个容器，并绘制图表

基础图表类型：线图，柱状图，饼图

配置项

						
							var myChart = echarts.init(document.getElementById('main'));
							var option = {
								title: {
									text: '一个小例子'
								},
			                    tooltip: {
			                        show: true
			                    },
			                    legend: {
			                        data:['销量']
			                    },
			                    /*直角坐标系grid中的x轴,一般情况下单个grid组件最多只能放左右两个x轴，多于两个x轴需要通过配置 offset属性防止同个位置多个x轴的重叠。
								*/
			                    xAxis : [
			                        {
			                            type : 'category', //类目轴，适用于离散的类目数据，为该类型时必须通过 data 设置类目数据。
			                            data : ["衬衫","羊毛衫","雪纺衫","裤子","高跟鞋","袜子"]
			                        }
			                    ],
			                    yAxis : [
			                        {
			                            type : 'value' //数值轴，适用于连续数据。
			                        }
			                    ],
			                    series : [
			                        {
			                            "name":"销量",
			                            "type":"bar",
			                            "data":[5, 20, 40, 10, 10, 20]
			                        }
			                    ]
			                };
			        
			                // 为echarts对象加载数据 
			                myChart.setOption(option);

事件

用在需要和某个图表上的事件进行交互的场合：

						
							myChart.on('click', function (params) {
							    window.open('https://www.baidu.com/s?wd=' + encodeURIComponent(params.name));
							});

其中的 params 参数涵盖了整个图表对象的信息（见官方文档）

使用中遇到的问题

1、容器高度固定，legend过多导致覆盖图表

解决方法：

  • 1.纵向排列；
  • 2.设置legend的大小以及 margin (配置项手册里面可以查到)；
  • 3.单独用一个DOM来放legend的内容，增加点击切换事件。

使用中遇到的问题：

2、单页面中需要展示多个图表

解决方法：

  • 重新定义容器myChart和option

使用中遇到的问题：

3、容器隐藏的情况下无法动态展示图表

解决方法：

  • 1.改display:none为visibility: hidden;
  • 2.明确指定div的宽高，然后在div显示后手动调用echartsInstance.resize调整尺寸。
